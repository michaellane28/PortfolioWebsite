A web application that converts color hex codes into code snippets for various programming languages

Users may choose from 6 different programming languages. Inputted hex codes are calculated based on the chosen language and the resulting code snippet is displayed.

Features

Input Verification:

Checks for '#' character and removes it from input if present.

Ensures that input is 6 characters. If not, site alerts user that input is invalid.

Smooth Transition:

Allows for easy navigation between each language.

Clears previous data when navigating through languages.

Code Display:

Code snippet texts are color coded with different colors for specific keywords and variable types.

Copy Functionality:

Allows users to easily copy code snippet to clipboard from the press of a button.
